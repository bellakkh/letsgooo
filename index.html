📅 How calendar export works
For Google Calendar:
Tap "Google Cal" next to any task → Google Calendar opens in a new tab → click Save. The event is added to your Google Calendar.

For Apple Calendar / Outlook:
Tap ".ics file" → a file downloads → open it → your calendar app asks to import it.

Export All:
The "Export all tasks" button downloads one .ics file containing all tasks that have a due date. You can import it into any calendar app at once.

Who can use this?
Anyone who opens the Bear tab on their device can export tasks to their own calendar independently — no shared account needed.

📅 Google Calendar sync
Two ways to add tasks to Google Calendar:
Option 1 — .ics file (works on any calendar)
1. Click 📅 .ics file on any task (must have a due date)
2. Open the downloaded file
3. Your calendar app opens and asks to import the event
4. Works with Google Calendar, Apple Calendar, Outlook and more

Export all tasks at once: click Export all (.ics) in the table header — downloads all tasks with due dates as one file.
Option 2 — Google Calendar direct link
1. Click Google Cal on any task
2. Google Calendar opens in your browser with the event pre-filled
3. Review and click Save — done!

Works for anyone who has the Bear tab open — each person adds it to their own Google Calendar.
💡 Tips
• Tasks need a due date set to be exportable
• Share this page with anyone who should see the tasks — they can export to their own calendar
• Priority (High/Medium/Low) is included in the calendar event title
• Comments and replies are included in the event description
